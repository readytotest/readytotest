# Welcome to My GitHub! 👋

Hey, welcome to my GitHub! I'm all about software testing. **I'm an exploratory tester at heart** with roots in UX and usability testing, but these repos are mostly focused on automated testing with tools like Playwright, Cypress, and CI/CD.

Everything here is hands-on experience. I learn by doing and I broke things a hundred different ways before figuring out how to get everything working.

Take a look around and hopefully you find something useful here, because I like sharing what I learn.

Want to know more or just see what else I'm up to? Here's my personal site:  
[readytotest.github.io](https://readytotest.github.io)
